Compensation Calculator

Switching to a new job can be hard and one of the hardest parts is making sure that your compensation is not being compromised by taking on a new position. This utility app is designed to help get all variables on the table to easily view what is your total compensation now and compare it to another offer.

To get started, please grab your latest paycheck so you can easily reference numbers from there.

All information entered is never stored either locally or on a server. Once you refresh this page, all entered data will disappear.

Formulas

Gross Salary

\[ \text{Gross Pay per Paycheck} * \text{Paychecks per Year} \]

Additional Compensation

\[ \text{Additional Compensation per Year} \]

Employer Match

\[ ( \text{Annual Gross Pay} + \text{Additional Compensation per Year} ) * \text{Employer Match (%)} \]

Paid Time Off/Holidays

\[ \text{Days Off per Year} * { \text{Annual Gross Pay} \over { ( 52 * \text{Days Expected to Work per Week} ) } } \]

Mileage

\[ 2 * \text{Miles one-way} * \text{Days Actually Worked On-Site per Year} * \text{Mileage Rate from IRS} \]

Time Driving

\[ 2 * \text{Minutes one-way} * { \text{1 hour} \over \text{60 minutes} } * { \text{Annual Gross Pay} \over \text{Expected Hours per Year} } * \text{Days Actually Worked On-Site per Year} \]

Total Compensation

\[ ( \text{Gross Pay} + \text{Additional Compensation} + \text{Annualized Employer Match (\$)} + \text{Paid Time Off (\$)} ) - ( \text{Mileage} + \text{Time Driving} ) \]
